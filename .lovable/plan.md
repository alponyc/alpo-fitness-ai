

# Alpo Fitness AI — Premium Mobile Dashboard

## Theme & Design System
- **Deep matte black** background (`#0A0A0A`), **crisp white** text, **bold red** accents (`#DC2626`)
- Mobile-first layout optimized for phone screens
- Glassmorphism effects for the AI chat bubble
- The uploaded Alpo Fitness logo will be used as a brand asset in the header area

## Components (top to bottom)

### 1. Top Navigation Bar
- "Alpo Fitness AI" header text with bold red accent on "Fitness"
- User profile avatar icon on the right

### 2. Workout History Card
- Dark card with "Workout Log" title
- **Yesterday**: "High Volume Legs (Scott Protocol)" with left/right arrows to browse days
- **Today**: "Active Recovery Cardio (The Flush) – 30 mins"

### 3. Personal Records (PR) Tracker
- Compact white card showing "Chest Press Max: 200 lbs"
- Red "Log New PR" button

### 4. Fridge Inventory Card
- Dark card titled "Fridge Inventory"
- List items: Churrasco (2 portions), Arabic Chicken (3 portions), Asparagus, Peppers, Onions, 42g Fairlife (1)
- Red "Scan Grocery Receipt" button

### 5. AI Accountability Chat Bubble
- Glassmorphism styled chat bubble with frosted glass effect
- Pre-filled insight message about the 198.6 weigh-in, sodium/water analysis, and meal recommendation
- Labeled "Executive Insight"

### 6. Injury Status Badge
- Red pulsing badge in the corner: "Elbow: Neutral Grip Protocol Only"
- Animated pulse effect to draw attention

### 7. Macro Rings
- Three circular progress rings at the bottom in red
- Protein: 142g, Carbs: 20g, Fats: 35g
- Clean labels beneath each ring

## Technical Notes
- All data is static/hardcoded for this initial build (no backend needed yet)
- Single-page scrollable mobile dashboard
- Smooth animations on the injury badge pulse and ring progress
